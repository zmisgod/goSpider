# fetch movie (www.dy2018.com)

因为[电影天堂](http://www.dy2018.com)的搜索做了一些防御导致搜索频率过高会出现搜索不出来的问题，所以我这边的解决办法是从百度搜索中搜索电影天堂中的电影。

## usage

```
git clone
cd /your/path/goSpider/fetchMovie/
go run fetchMovie.go [电影名称]
```
